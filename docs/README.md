## Resumen

Rental Manager permite a administradores y usuarios gestionar una flota de vehículos, registrar personas (conductores/propietarios), crear contratos de alquiler u opción de compra, llevar un control diario de ingresos y gastos, y visualizar reportes semanales, mensuales y personalizados.

**Moneda:** COP (pesos colombianos) — todos los valores monetarios.
